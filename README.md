# FeatureGate React

Production-style React application demonstrating scalable state management using Context API for feature toggling and Redux Toolkit for async business data handling.

---

# Features

## React Context API
- Feature toggling
- Global UI state management
- Dark mode support
- Context Provider pattern
- Custom hooks

## Redux Toolkit
- Centralized state management
- Async API handling
- Redux slices
- Loading and error states
- Scalable business data architecture

## Dashboard Features
- User management dashboard
- Analytics feature toggle
- Dynamic UI rendering
- Modular component structure

---

# Tech Stack

- React.js
- Redux Toolkit
- React Context API
- React Redux
- JavaScript
- Vite

# State Management Architecture

## Context API Responsibilities

Used for lightweight global UI state:

- Feature toggles
- Theme settings
- UI preferences
- Admin feature visibility

## Redux Toolkit Responsibilities

Used for scalable business and async state:

- API data handling
- User management
- Loading states
- Error handling
- Centralized application data

# Getting Started

## Clone Repository

```bash
git clone https://github.com/vish-git/featuregate-react.git
```

## Navigate to Project

```bash
cd featuregate-react
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

# Example Features
